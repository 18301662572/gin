# gin框架
使用gin构建了企业级脚手架，代码简洁易读，可快速进行高效web开发。
本项目是针对gin框架进行初级分类学习

### gin框架中文文档（详解）： https://www.jianshu.com/p/98965b3ff638/

###文件分层
```
├── middleware   gin中间件
├── other 其他
│   └── other_autotis 自动创建证书
    └── other_shutdown 优雅的关停服务器
    └── other_template 模板渲染
├── param 参数
├── router 路由
├── start 初识gin
├── vaild 验证
└── gin.png gin基础框架图

```
